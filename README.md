# sparkdigital

Spark Digital project.
## Development Setup
### Prerequisites

* XCode for ios app
* Android Studio for Android app
* [fvm](https://github.com/leoafarias/homebrew-fvm)
### Initial setup

* Setup the necessary [environment variables][environment-variables-setup].
* Run `fvm install`

### Run app

* Run the emulator/simulator
* Run `fvm fluter run`
